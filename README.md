HR Analytics SQL Project
Overview

This project analyzes employee attrition and performance using SQL based on the IBM HR Analytics Employee Attrition & Performance dataset. The goal is to answer real-world HR business questions using basic to intermediate SQL queries.

Objectives:
Analyze employee attrition across departments and job roles
Understand the impact of overtime and work-life balance
Compare income and satisfaction levels of employees

Tools Used:
SQL (MySQL)

Dataset:
IBM HR Analytics Employee Attrition & Performance
1,470 employee records
